新增獎助學金 Create

若填寫，系統會建立獎學金與機構的關聯。

若選擇特定身份，則僅該身份的學生可以看到並申請此獎學金。

小提示

1. 設定申請開始和截止日期，學生只能在此期間申請。

2. 「建立後立即發放」會將獎學金狀態設為已發放，學生即可查看與申請。

3. 機構 ID 可在「帳號管理」中查詢，格式通常如 ORG001。

編輯獎助學金 Edit

獎學金名稱不可修改

若填寫，系統會更新獎學金與機構的關聯。留空則移除關聯。

編輯說明

1. 獎學金名稱建立後無法修改。

2. 修改提供機構 ID 可補充或更新獎學金的提供單位資訊。

3. 留空機構 ID 將移除該獎學金與機構的關聯。
